# Welcome to star

star is an accessible document reader that speaks almost anything aloud and
highlights each word as it reads. No cloud account, no internet required.

You are reading this welcome page **as a real document** — so every control
works right here. Go ahead and try them on this text.

## Hear it read aloud

Press **Space** to start reading from the top. Press **Space** again to pause,
and **Escape** to stop. Use **−** and **+** (or the toolbar's Slower and Faster
buttons) to change the speaking rate while it reads.

## Move around freely

The text cursor moves with the **arrow keys** — by character, or by word with
**Ctrl**. Hold **Shift** while moving to select a passage, then press **Ctrl+D**
to look up the word under the cursor. Caret browsing is on by default; toggle it
any time with **F7**.

Jump by structure too: previous and next **sentence**, **paragraph**, and
**heading** all have toolbar buttons (hover any button to see what it does and
its keyboard shortcut).

## Open your own documents

- **Ctrl+O** — open a file: PDF, Word, EPUB, PowerPoint, HTML, Markdown,
  spreadsheets, DAISY, and many more.
- **Ctrl+Shift+O** — open a web page by URL.
- **Ctrl+Shift+L** — point star at any folder (for example a synced Dropbox or
  OneDrive folder) and browse it as a library.

## Make it yours

- **F5** cycles the color theme (Obsidian is the default; Zed and high-contrast
  themes are included).
- Change the display font, line spacing, and reading aids from the **View** menu.
- Highlight a selection, take notes, and see your reading level — all from the
  toolbar or the menus.

## Get help any time

Press **F1** to open the full README — the complete reference for every feature,
keyboard shortcut, and setting. It opens right here as a document, so you can
read it aloud and search it just like this page.

Happy reading.
