# Write a document and export it

star is also a Markdown authoring tool: create a document from scratch, format
it, and export it to PDF, audio, Braille, and more. This is a GUI walkthrough —
**nothing to run here**; follow along in the app.

**You'll need:** `star` installed. Launch it with `star`.

## Walkthrough

1. **New document** — **File ▸ New** (`Ctrl+N`). star opens a blank document in
   **edit mode** with a **live preview** beside the source, split evenly.
2. **Type and format** — write Markdown, or use the **Format** menu / toolbar:
   **Bold** (`Ctrl+B`), *Italic* (`Ctrl+I`), headings, lists, **Insert Table…**,
   **Insert Image…**, and **Insert Link** (`Ctrl+K`). The preview updates as you
   type.
3. **Find & replace** — **Edit ▸ Find & Replace…** (or `Ctrl+F`, then the
   **Replace ▾** row) to change text, one match or all.
4. **Dictate instead of type** — **Tools ▸ Dictate Note** records your voice and
   transcribes it offline (star fetches the speech-to-text engine on first use).
5. **Save** — **`Ctrl+S`** saves and *keeps you editing* (the first save asks
   where); **`Ctrl+E`** finishes and returns to reading. Unsaved work is
   autosaved and offered back if star ever closes unexpectedly.
6. **Export** — **File ▸ Export ▸** Markdown / PDF / Braille (BRF) / Audio /
   Subtitles / Audiobook. Exports use the **live editor buffer**, so you can
   export a draft without saving first.

## What makes it useful

- A **real live-edit loop** — format, preview, save, keep going.
- **Export to accessible formats** — Braille and spoken audio, not just PDF.
- **Autosave & crash recovery** so a new, unsaved document is never lost.
